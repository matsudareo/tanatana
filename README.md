# Tanatana（棚卸計上アプリ）
PHP自作
# 概要
お店で月末に行う棚卸業務を簡単に行うための計上アプリを作成しました。

管理者ユーザーと一般ユーザーを分け、それぞれでログインできるようにしています。

# 使い方
管理者ユーザー

ファイルの作成、商品の登録・編集・削除、商品一覧をカテゴリー別で見れるようにしています。
